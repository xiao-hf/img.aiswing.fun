Overview

流式生图 API

直接请求下面任意一条线路。接口返回 SSE 流，图片数据为 base64。

MethodPOST
Path/v1/images/generations/events
AuthAuthorization: Bearer sk-your-key
Accepttext/event-stream

注意：cdn/gpt/codex 的 /v1/images/generations/events 由 Aiswing 代理转成 Responses 流式生图请求，因此可直接用本文档的请求体。

Text to Image

文生图：curl 示例

推荐 cdn 线路 
curl --location 'https://cdn.aiswing.fun/v1/images/generations/events' \
  --header 'Authorization: Bearer sk-your-key' \
  --header 'Content-Type: application/json' \
  --header 'Accept: text/event-stream' \
  --data '{
    "model": "gpt-image-2",
    "prompt": "A premium product poster, clean studio lighting, no text, no watermark",
    "size": "3840x2160",
    "quality": "high",
    "format": "png"
  }'

文生图不需要 reference_images。4K 横图用 3840x2160，4K 竖图用 2160x3840。

Image to Image

图生图：携带参考图

reference_images 
curl --location 'https://cdn.aiswing.fun/v1/images/generations/events' \
  --header 'Authorization: Bearer sk-your-key' \
  --header 'Content-Type: application/json' \
  --header 'Accept: text/event-stream' \
  --data '{
    "model": "gpt-image-2",
    "prompt": "Keep the same subject, turn it into a cinematic product poster",
    "size": "3840x2160",
    "quality": "high",
    "format": "png",
    "reference_images": [
      "data:image/png;base64,iVBORw0KGgoAAA..."
    ]
  }'

参考图推荐使用 data:image/png;base64,...。浏览器中可用 FileReader.readAsDataURL(file) 转换本地图片。

请求参数说明

model模型

推荐 gpt-image-2。

prompt提示词

必填。描述主体、风格、构图、光线、颜色，以及不要出现的文字或水印。

size分辨率

支持 1024x1024、1536x1024、1024x1536、3840x2160、2160x3840、2880x2880、auto。

quality质量

可选 auto、low、medium、high。4K 或正式出图建议 high。

format输出格式

可选 png、jpeg、webp。

reference_images参考图

可选数组。不传就是文生图；传入后就是图生图。支持 base64 data URL 或公网图片 URL。

SSE 流式返回

response.image_generation_call.partial_image预览图

生成过程中的中间图，读取 partial_image_b64。

response.completed最终图

完成事件。最终图通常在 response.output[].result。

[DONE]结束

SSE 流结束标记。

JavaScript 流式读取示例

const endpoint = "https://cdn.aiswing.fun/v1/images/generations/events";
const response = await fetch(endpoint, {
  method: "POST",
  headers: {
    "Authorization": "Bearer sk-your-key",
    "Content-Type": "application/json",
    "Accept": "text/event-stream"
  },
  body: JSON.stringify({
    model: "gpt-image-2",
    prompt: "4K product poster, clean studio lighting",
    size: "3840x2160",
    quality: "high",
    format: "png",
    reference_images: []
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();
let buffer = "";
while (true) {
  const { value, done } = await reader.read();
  if (done) break;
  buffer += decoder.decode(value, { stream: true });
  const chunks = buffer.split("\n\n");
  buffer = chunks.pop() || "";
  for (const chunk of chunks) {
    const line = chunk.split("\n").find((x) => x.startsWith("data:"));
    if (!line) continue;
    const raw = line.slice(5).trim();
    if (raw === "[DONE]") break;
    const event = JSON.parse(raw);
    const b64 = event.b64_json || event.partial_image_b64 || event.response?.output?.find?.((x) => x.result)?.result;
    if (b64) document.querySelector("img").src = `data:image/png;base64,${b64}`;
  }
}
