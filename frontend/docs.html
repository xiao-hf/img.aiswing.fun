Aiswing API Documentation

cdn.aiswing.fun 流式生图 API 文档

本页描述直接调用 https://cdn.aiswing.fun/v1/responses 的流式生图接口。 请求使用 Responses API 结构，工具类型为 image_generation，支持文生图、4K 尺寸和携带参考图。

目录 快速开始 请求头 请求体结构 参数说明 携带参考图 流式响应 错误处理

快速开始

POSThttps://cdn.aiswing.fun/v1/responses

最小可用请求：传入提示词，启用 stream: true，并声明 image_generation 工具。

curl --location 'https://cdn.aiswing.fun/v1/responses' \
  --header 'Authorization: Bearer sk-your-cdn-key' \
  --header 'Content-Type: application/json' \
  --header 'Accept: text/event-stream' \
  --data '{
    "model": "gpt-image-2",
    "stream": true,
    "tools": [
      {
        "type": "image_generation",
        "size": "1024x1024",
        "quality": "high",
        "output_format": "png"
      }
    ],
    "input": [
      {
        "role": "user",
        "content": [
          {
            "type": "input_text",
            "text": "都市暴雨夜，一个年轻外卖员穿黄色雨衣站在霓虹街头，电影感光影，不要文字，不要水印"
          }
        ]
      }
    ]
  }'

请求头

名称必填说明
Authorization是格式：Bearer sk-your-cdn-key，使用 CDN 地址对应的 Key。
Content-Type是固定为 application/json。
Accept建议建议传 text/event-stream，表示接收 SSE 流式事件。

请求体结构

接口不是 multipart/form-data；即使携带参考图，也需要把图片转成 base64 data URL 后放进 JSON。

{
  "model": "gpt-image-2",
  "stream": true,
  "tools": [
    {
      "type": "image_generation",
      "size": "3840x2160",
      "quality": "high",
      "output_format": "png"
    }
  ],
  "input": [
    {
      "role": "user",
      "content": [
        { "type": "input_text", "text": "你的提示词" }
      ]
    }
  ]
}

参数说明

参数类型说明
modelstring模型名。当前页面默认使用 gpt-image-2。
streamboolean是否流式返回。生图建议固定传 true。
inputarray对话输入数组。每项通常包含 role: "user" 和 content。
input[].rolestring消息角色，常用 user。
input[].contentarray内容数组，可同时放文本和参考图。
content[].typestringinput_text 表示提示词文本，input_image 表示参考图。
content[].textstring当 type 为 input_text 时填写提示词。
content[].image_urlstring当 type 为 input_image 时填写图片 URL 或 base64 data URL，例如 data:image/png;base64,...。
toolsarray工具数组。生图必须包含一个 image_generation 工具。
tools[].typestring固定为 image_generation。
tools[].sizestring输出尺寸。常用：1024x1024、1536x1024、1024x1536、3840x2160、2160x3840、2880x2880。
tools[].qualitystring质量档位。常用：auto、low、medium、high。
tools[].output_formatstring图片格式。常用：png、jpeg、webp。
tools[].output_compressionnumber压缩质量，通常用于 jpeg 或 webp，范围建议 0-100。

携带参考图

参考图推荐转成 base64 data URL，和提示词一起放入 input[].content。下面示例会读取本地图片并发起流式请求。

import fs from "node:fs";

const apiKey = process.env.AISWING_CDN_KEY;
const imageBase64 = fs.readFileSync("./reference.png").toString("base64");

const response = await fetch("https://cdn.aiswing.fun/v1/responses", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${apiKey}`,
    "Content-Type": "application/json",
    "Accept": "text/event-stream"
  },
  body: JSON.stringify({
    model: "gpt-image-2",
    stream: true,
    tools: [
      {
        type: "image_generation",
        size: "1024x1024",
        quality: "high",
        output_format: "png"
      }
    ],
    input: [
      {
        role: "user",
        content: [
          { type: "input_text", text: "保留人物姿势，改成赛博朋克电影海报风格，不要文字，不要水印" },
          { type: "input_image", image_url: `data:image/png;base64,${imageBase64}` }
        ]
      }
    ]
  })
});

流式响应

响应为 SSE 文本流。生成过程中可能返回进度事件和预览图，完成时返回最终图片 base64。

事件 type关键字段说明
response.created-任务已创建。
response.in_progress-任务处理中。
response.image_generation_call.generating-图片正在生成。
response.image_generation_call.partial_imagepartial_image_b64预览图 base64，可直接显示或保存。
response.completedresponse.output[].result最终图片 base64。通常取 type=image_generation_call 的 result 字段。
errorerror.message生成失败或请求参数错误。
// SSE data 示例
{
  "type": "response.completed",
  "response": {
    "output": [
      {
        "type": "image_generation_call",
        "result": "iVBORw0KGgoAAAANSUhEUg..."
      }
    ]
  }
}

错误处理

  • 401 Unauthorized：Key 无效、额度不足、未传 Authorization，或使用了错误地址对应的 Key。
  • 404 Not Found：路径错误。流式生图应请求 /v1/responses，不是旧的 /v1/images/generations。
  • Failed to fetch：浏览器跨域、网络中断或 CDN 超时。生产环境建议由后端代理请求并保存任务结果。
  • stream ended without image data：上游流结束但没有返回图片，检查模型、尺寸、提示词和账号权限。
  • 图片太大：参考图 base64 会显著增大请求体，建议压缩后再传，并控制后端 MAX_BODY_BYTES。
