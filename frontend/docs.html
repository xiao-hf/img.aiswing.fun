Aiswing Image API

流式生图 API

直接请求下面任意一条线路。复制 curl 后只需要替换 sk-your-key，即可开始生图。

Quick Start

curl 示例：文生图

推荐 cdn 线路 
curl --location 'https://cdn.aiswing.fun/v1/images/generations/events' \
  --header 'Authorization: Bearer sk-your-key' \
  --header 'Content-Type: application/json' \
  --header 'Accept: text/event-stream' \
  --data '{
    "model": "gpt-image-2",
    "prompt": "A tiny clean app icon, blue gradient sphere, white background",
    "size": "1024x1024",
    "quality": "high",
    "format": "png"
  }'

把 sk-your-key 替换成你的 API Key。接口返回 SSE 流，图片数据是 base64。

三条线路

cdnhttps://cdn.aiswing.fun/v1/images/generations/events默认推荐线路
gpthttps://gpt.aiswing.fun/v1/images/generations/events备用线路
codexhttps://codex.aiswing.fun/v1/images/generations/events备用线路

请求参数说明

model模型

推荐 gpt-image-2。

prompt提示词

必填。描述你要生成或编辑的画面，例如主体、风格、构图、光线、颜色、不要出现的文字等。

size分辨率

支持 1024x1024、1536x1024、1024x1536、3840x2160、2160x3840、2880x2880、auto。4K 横图用 3840x2160，4K 竖图用 2160x3840。

quality质量

可选 auto、low、medium、high。4K 或正式出图建议 high。

format输出格式

可选 png、jpeg、webp。透明或高保真建议 png，网页体积优先可用 webp。

reference_images参考图

可选数组。不传就是文生图；传入后就是图生图。推荐传 data:image/png;base64,...，也可传公网图片 URL。

curl 示例：带参考图

curl --location 'https://cdn.aiswing.fun/v1/images/generations/events' \
  --header 'Authorization: Bearer sk-your-key' \
  --header 'Content-Type: application/json' \
  --header 'Accept: text/event-stream' \
  --data '{
    "model": "gpt-image-2",
    "prompt": "Keep the same subject, turn it into a cinematic product poster",
    "size": "3840x2160",
    "quality": "high",
    "format": "png",
    "reference_images": [
      "data:image/png;base64,iVBORw0KGgoAAA..."
    ]
  }'

浏览器中可用 FileReader.readAsDataURL(file) 把本地图片转成 data:image/...;base64。

流式返回怎么读

response.image_generation_call.partial_image预览图

生成过程中的中间图，读取 partial_image_b64。

response.completed最终图

完成事件，最终图通常在 response.output[].result。

[DONE]结束

SSE 流结束标记。

const imgSrc = `data:image/png;base64,${base64}`;

JavaScript 流式读取示例

const endpoint = "https://cdn.aiswing.fun/v1/images/generations/events";
const response = await fetch(endpoint, {
  method: "POST",
  headers: {
    "Authorization": "Bearer sk-your-key",
    "Content-Type": "application/json",
    "Accept": "text/event-stream"
  },
  body: JSON.stringify({
    model: "gpt-image-2",
    prompt: "4K product poster, clean studio lighting",
    size: "3840x2160",
    quality: "high",
    format: "png",
    reference_images: []
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();
let buffer = "";
while (true) {
  const { value, done } = await reader.read();
  if (done) break;
  buffer += decoder.decode(value, { stream: true });
  const chunks = buffer.split("\n\n");
  buffer = chunks.pop() || "";
  for (const chunk of chunks) {
    const line = chunk.split("\n").find((x) => x.startsWith("data:"));
    if (!line) continue;
    const raw = line.slice(5).trim();
    if (raw === "[DONE]") break;
    const event = JSON.parse(raw);
    const b64 = event.partial_image_b64 || event.response?.output?.find?.((x) => x.result)?.result;
    if (b64) document.querySelector("img").src = `data:image/png;base64,${b64}`;
  }
}
